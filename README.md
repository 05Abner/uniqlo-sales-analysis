# 优衣库销售数据分析与运营策略建议
## 项目背景
本项目基于优衣库线下门店销售数据，从**电商运营与产品决策**视角出发，通过数据清洗、多维度分析、交叉透视及可视化看板，提炼出可落地的运营策略，为门店的商品组合、区域营销、旺季备货等提供决策支持。
## 技术栈
- **数据处理**：Python (Pandas, Matplotlib) + SQLite
- **分析工具**：Excel 数据透视表（手动交叉分析）
- **可视化**：Power BI 仪表板 + Matplotlib 图表
## 分析流程
1. **数据清洗与预处理**：使用Pandas处理缺失值与异常值，统一字段格式，删除重复数据，保证数据质量。
2. **数据存储**：将清洗后数据写入SQLite数据库，便于结构化查询。
3. **多维度探索分析**通过Python从**城市、产品、用户、时间**四大维度统计销售额、销量、客户数等核心指标，并生成 Matplotlib 可视化图表。
4. **交叉深度洞察（手动 Excel 透视表）**：将清洗后的数据导出，在 Excel 中手动创建透视表，完成「城市×品类」「年龄×性别」「月份×品类」等交叉分析，发现高价值组合与波动规律。
5. **可视化看板**：用 Power BI 搭建销售仪表板，包含 KPI 卡片、门店对比、品类结构、趋势分析，方便运营团队快速监控。
## 核心业务结论与运营建议
### 1. 区域结构
- 深圳贡献 53.7% 的销售额（17.3万），是杭州的 2.8 倍、西安的 5.7 倍，区域极度不均衡。  
**→ 建议**：深圳为核心仓，优先保障 T 恤库存，主推「T恤+配件」组合；西安以爆款 T 恤引流，搭配小件配件提升客单价。
### 2. 品类结构
- T 恤占销售额 41.6%（13.4万），为绝对核心品类；配件排名第三（14.6%），具备高连带潜力。  
**→ 建议**：门店陈列和线上推荐强化「T恤+配件」搭配；杭州、重庆等区域重点提升配件连带率。
### 3. 用户画像
- 女性占比 71.2%，消费力为男性的 2.5 倍；30-34 岁客群最强（25.3%），25-39 岁合计占 70.8%。  
**→ 建议**：聚焦 30-39 岁女性做精准营销，如会员日活动、专属优惠券；新品及联名款优先触达该群体。
### 4. 时间规律
- 8 月单月贡献 63% 的销售额，呈现年度爆发；周五为周内高峰（占 22.1%），周三仅为 1.6%。  
**→ 建议**：8 月前提前备货 T 恤、配件、当季新品；周五加大促销预热；淡季（1-3 月、11 月）通过会员日或清仓活动提升客流。
### 5. 渠道现状
- 数据中 100% 为线下销售，线上渠道空白。  
**→ 建议**：可探索小程序商城或社群电商，将线下高连带组合同步至线上，拓展增量。
