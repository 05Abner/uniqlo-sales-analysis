# 优衣库线下门店销售数据分析项目
基于 Python + Excel + PowerBI的全流程零售数据分析与业务洞察

# 项目简介
本项目针对优衣库线下门店销售数据，构建了从数据清洗、数据库存储、多维度统计分析到可视化展示的完整分析闭环。
旨在通过数据挖掘，分析各门店、产品、用户群体的销售表现及时间趋势，为线下门店运营管理、货品调配和营销活动提供数据支撑。

# 技术
- **编程语言**：Python (Pandas, Matplotlib)
- **数据库**：SQLite
- **分析工具**：Excel (数据透视表)
- **开发环境**：PyCharm / MacOS

# 项目文件结构
文件名称 & 文件描述 
**uniqlo_sales_analysis.py**  核心分析代码（数据清洗、可视化、SQL写入） 
**优衣库销售数据.xlsx** 原始数据源 
**优衣库_清洗后数据.xlsx**  预处理完成的干净数据集 
**优衣库销售数据分析.xlsx**  Excel 透视分析报告（产品×城市、性别×年龄、月份×产品） 
**uniqlo_sales.db** 结构化数据存储文件 
**可视化图表/*.png** 5张核心分析图谱（城市/产品/性别/年龄/月度）, PowerBI 可视化看板

# 分析流程
1. **数据清洗**：处理缺失值与异常值，统一数据格式，保证数据质量。
2. **数据存储**：将清洗后的数据写入 SQLite 数据库，完成结构化持久化。
3. **探索性分析**：基于 Python 完成销售额、销量、等基础指标统计。
4. **交叉分析**：利用 Excel 数据透视表进行深度拆解。
5. **可视化呈现**：利用PowerBI生成专业图表，直观展示分析结果并输出业务结论。

# 核心业务结论
1. **门店维度**：一线城市及核心商圈门店贡献主要营收，需重点维护供应链稳定性。
2. **产品维度**：基础款产品表现稳健，季节性产品销售波动较大，需关注库存周转。
3. **用户维度**：女性客群（25-35岁）是绝对主力，建议针对该群体推出限定款系列。
4. **时间维度**：存在明显的季节周期特征，淡季可通过会员日活动提升客流。

## 可视化成果展示
- **各城市销售额排行**：快速定位核心营收区域
- **产品类别销售占比**：优化门店货品陈列结构
- **性别/年龄群体对比**：精准定位目标用户偏好
- **月度销售趋势**：预测季节性波动，辅助备货计划
